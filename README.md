# Web Traffic Analytics

## Objective

Analyze website traffic data to understand user behavior and improve engagement.

## Metrics Analyzed

- Page Views
- Session Duration
- Bounce Rate
- User Journey
- Drop-off Points

## Technologies

- Python
- Pandas
- GitHub

## Dataset Fields

- UserID
- Page
- PageViews
- SessionDuration
- Bounce
- NextPage

## Run

```bash
pip install -r requirements.txt
python web_traffic_analysis.py
```

## Output

- Traffic statistics
- Bounce rate
- Most viewed page
- User journey analysis
- Recommendations
